### Quick Setup
#### Clone Repository
git clone https://github.com/your-username/nexus-ai-chatbot.git
cd nexus-ai-chatbot

#### Backend Setup
cd backend
npm install

Create .env file
PORT=5000 

MONGO_URI=your_mongo_uri

GEMINI_API_KEY=your_api_key

Start Server
node server.js

#### Frontend Setup
cd ../Frontend

npm install

npm run dev
